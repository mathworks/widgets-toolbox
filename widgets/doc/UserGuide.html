Widgets Toolbox User Guide

Widgets Toolbox helps you efficiently develop advanced user interfaces in MATLAB and App Designer. Widgets combine existing control functionalities together into larger, reusable, common functionality to accelerate development of graphical user interfaces.
Widgets Toolbox also will enable you to develop efficiently if your app needs would benefit from advanced, hand-coded architecture. Reusable object-oriented content provides functionality for managing user session files, preferences, and context pane switching.

Widgets

Widgets are based on MATLAB's Custom UI Component functionality using the ComponentContainer class. Multiple UI controls are connected together in reusable and common "building blocks" that enable accelerated development. The widgets may be used in App Designer apps or in hand-coded apps.

Advanced Application Development Components

If you have experience with app development and object-oriented programming, you can work more efficiently developing advanced apps if you use a modular hand-code approach. You can use model-view-controller or a similar separated presentation architecture.
If you develop multiple apps, you will find there are repetitive common needs that are more advanced than the capabilities in App Designer. Using an object-oriented framework enables you to more flexibility to inherit and reuse common capabilities and reduce development effort.
