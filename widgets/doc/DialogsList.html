Listing of Included Dialogs

The dialogs included with Widgets Toolbox are shown below. These dialogs reside inside of the figure window for compatibility with web apps.

List Selection Dialog

The wt.dialog.ListSelection dialog requests the user to choose from a list of items. This is similar to listdlg.
Create a figure:
fig = uifigure("Position",[100 100 400 400]);
fig.Name = "List Selection Dialog Example";
Create the dialog:
dlg = wt.dialog.ListSelection(fig);
dlg.Title = "Purchase Fruit";
dlg.Prompt = "Make your selection:";
dlg.Items = ["Apple","Banana","Cherry","Grape","Orange","Pineapple"];
You can wait for output using this syntax:
% [output, lastAction] = dlg.waitForOutput()

Login Dialog

The wt.dialog.Login dialog prompts a user to provide their username and password credentials.
Create a figure:
fig = uifigure("Position",[100 100 400 400]);
fig.Name = "Login Dialog Example";
Create the dialog:
dlg = wt.dialog.Login(fig);
dlg.Title = "Please log in";
You can wait for output using this syntax:
% [output, lastAction] = dlg.waitForOutput()
